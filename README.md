# DWC - Decentralized Wallet Checker

DWC is a webplatform to check for decentralized wallets, his balances and transactions.

### Getting Started:

To set this projetct locally, first install dependencies:

```
yarn install
```

and run:

```
yarn dev
```

Access localhost:3000
